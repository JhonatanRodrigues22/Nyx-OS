# @nyx-os/scheduler

Scheduler oficial do Nyx OS.

Este pacote define o contrato `NyxScheduler` e a implementação inicial `SchedulerManager`, responsáveis por registrar e executar tarefas recorrentes em memória.

## Escopo

- Registro de tarefas recorrentes.
- Start, stop, pause e resume.
- Execução por intervalo.
- Eventos técnicos no Event Bus oficial.
- Estado em memória das tarefas.

## Fora do escopo

- Persistência.
- Automations.
- Memory.
- IA.
- Skills.
- Cron avançado.
- Distribuição entre processos.
