# @nyx-os/plugin

Framework oficial de plugins do Nyx OS.

Este pacote define o contrato `NyxPlugin` e o `PluginManager`, responsáveis por registrar, inicializar, descartar, remover e consultar plugins de forma desacoplada.

## Escopo

- Contrato oficial de plugin.
- Manager em memória.
- Estados de lifecycle.
- Eventos técnicos emitidos pelo Event Bus oficial.

## Fora do escopo

- Descoberta automática de plugins.
- Plugins externos.
- Scheduler.
- Memory.
- IA.
- Skills.
- Automation.
