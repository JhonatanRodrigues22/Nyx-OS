# @nyx-os/local-gateway

Servidor WebSocket local e bridge de capabilities remotas do Nyx OS.

O pacote fornece contratos versionados, autenticação de handshake, heartbeat, registry de instâncias, correlação de comandos e adapters para o `CapabilityManager` e o `ToolManager` oficiais.

Consulte [Fundação de Comunicação Local](../../docs/arquitetura/local-communication-foundation.md) e [ADR-0024](../../docs/decisoes/ADR-0024-websocket-envelope-json-local.md).
