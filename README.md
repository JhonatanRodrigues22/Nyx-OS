# Nyx OS

Nyx OS is a personal operating system aimed at centralizing the capture, storage and retrieval of your day-to-day information with minimal friction. This repository starts empty: an assistant called **Nyx Programa** will generate the project structure and code based on architecture instructions stored in the `.ai` folder. See `.ai/nyx_os_architecture.md` for details.

The repository excludes personal documentation and scripts: the `Fundamentos` folder (which contains architecture and guideline documents) is not versioned. Environment files (`.env*`), build outputs (`dist/`), scripts (`scripts/`), private docs (`Fundamentos/`) and virtual environments (`.venv/`) are ignored by `.gitignore`.

Please do not add confidential information or credentials to this repository. Use environment variables and the `Fundamentos` folder (excluded) for personal notes.
