# @nyx-os/web

Aplicação web inicial do Nyx OS.

## Responsabilidade

- Entregar a experiência visual do Nyx OS.
- Hospedar rotas web e APIs iniciais enquanto o projeto ainda é simples.
- Servir como cockpit/dashboard inicial.

## Comandos

Use os comandos pela raiz do monorepo:

```bash
npm run dev
npm test
npm run lint
npm run build
```

## Observação

Esta aplicação não deve concentrar toda a lógica de domínio no longo prazo. Regras compartilhadas devem migrar para `packages/*` quando existirem.
