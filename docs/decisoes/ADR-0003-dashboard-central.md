# ADR-0003: Dashboard como centro da experiência

## Status

Aceita.

## Contexto

O Nyx OS deve funcionar como uma casa digital e central pessoal. Para isso, o usuário precisa de um ponto de entrada claro.

## Decisão

O dashboard será o centro inicial da experiência do Nyx OS.

Ele deve reunir contexto do dia, tarefas, projetos, check-ins, hábitos e informações importantes.

## Consequências

- O dashboard deve priorizar ação e contexto, não apenas estética.
- Módulos importantes devem conseguir fornecer dados para o dashboard.
- O MVP deve ter um dashboard simples, mesmo que ainda não seja visualmente final.
- A evolução visual deve vir depois da utilidade operacional.
