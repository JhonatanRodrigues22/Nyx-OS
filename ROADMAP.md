# Roadmap do Nyx OS

## Sprint 0 — Constituicao e Fundamentos

- Definir a visao do Nyx OS.
- Registrar principios, stack inicial e entidades centrais.
- Criar a fundacao tecnica do projeto.

## Sprint 0.5 — Limpeza, Arquitetura e Controle

- Reorganizar documentacao.
- Separar fonte de verdade conceitual de prompts de IA.
- Corrigir `.gitignore`.
- Versionar scripts auxiliares uteis.
- Validar que a aplicacao continua funcional.

## Sprint 1 — Base funcional da interface

- Consolidar shell visual inicial.
- Definir navegacao basica.
- Preparar componentes reutilizaveis minimos.

## Sprint 2 — Dashboard inicial

- Criar visao inicial do dia.
- Mostrar estados basicos e proximas acoes.
- Preparar espacos para tarefas, projetos e check-ins.

## Sprint 3 — Captura e organizacao de informacoes

- Implementar captura rapida.
- Organizar tarefas, notas, decisoes e entradas financeiras.
- Definir validacoes e contratos de API para criacao de registros.

## Sprint 4 — Integracao com Nyx Local

- Definir contrato de comunicacao com o ambiente local.
- Preparar sincronizacao ou ponte controlada entre local e nuvem.

## Sprint 5 — Automacao e assistente pessoal

- Integrar automacoes seguras.
- Preparar comandos da assistente pessoal.
- Registrar auditoria de acoes automatizadas.
